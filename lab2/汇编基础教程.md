## 第 1 部分：什么是汇编语言？

### 计算机如何工作的三个层次

```
┌─────────────────────────────────┐
│  高级语言 (C++, Python, Java)   │  ← 我们平时写的代码
├─────────────────────────────────┤
│    编译器翻译                   │  ← 把高级语言转成汇编
├─────────────────────────────────┤
│      汇编语言 (Assembly)        │  ← CPU 能理解的指令
├─────────────────────────────────┤
│     机器码 (0101...)            │  ← CPU 直接执行
└─────────────────────────────────┘
```

**汇编的作用**：
- 直接控制 CPU 和内存（比 C++ 更底层）
- 性能最优（编译器有时无法做得更好）
- 理解计算机的本质（寄存器、内存、指令执行）

### 为什么要学汇编？

在这个实验中，你需要用汇编来：
1. **直接读写内存**（没有中间层）
2. **实现分支和循环**（if/else, while）
3. **调用函数**（传参、返回值）
4. **理解 C++ 代码如何工作**

## 第 2 部分：CPU 寄存器入门

### 什么是寄存器？

寄存器是 CPU 内部的**超高速临时存储**。与内存相比：

```
内存    ：慢，容量大（GB级），长期存储
寄存器  ：快，容量小（8个32位），临时计算用
```

### 8 个通用寄存器（32 位）

在 32 位 x86 中，有 8 个通用寄存器：

| 寄存器 | 用途 | 特殊说明 |
|--------|------|---------|
| **eax** | 累加器 | 乘法、除法、返回值放这里 |
| **ebx** | 基址寄存器 | 存储地址的基准 |
| **ecx** | 计数器 | loop 指令默认用这个 |
| **edx** | 数据寄存器 | 配合 eax 做乘除 |
| **esp** | 栈指针 | ⚠️ **不能随意修改** |
| **ebp** | 帧指针 | ⚠️ **通常不用** |
| **esi** | 源指针 | 源地址指针 |
| **edi** | 目标指针 | 目标地址指针 |

### 寄存器的"缩写版本"

同一个寄存器可以用不同大小访问：

```
eax (32位)
 ├─ ax (低 16 位)
 │   ├─ ah (高 8 位)
 │   └─ al (低 8 位)
```

**例子**：
```asm
mov eax, 0x12345678      ; eax = 0x12345678
mov ax, 0x1234           ; eax 变成 0x12341234（只改了低 16 位）
mov al, 0x12             ; eax 变成 0x12341212（只改了最低 8 位）
```

## 第 3 部分：最重要的汇编指令

### 1️⃣ 数据传送：mov

**作用**：复制数据从一个地方到另一个地方

```asm
mov <目的>, <源>         ; 目的 = 源
```

**例子**：
```asm
mov eax, 10              ; eax = 10（常数赋值）
mov ebx, eax             ; ebx = eax（寄存器间赋值）
mov ecx, [esi]           ; ecx = 内存中地址 esi 处的值（读内存）
mov [edi], eax           ; 内存地址 edi 处 = eax（写内存）
```

**⚠️ 注意**：
- `mov [地址], [地址]` ❌ 不允许（内存到内存不行）
- 如需内存→内存，必须借助寄存器：
  ```asm
  mov eax, [esi]           ; 先读到 eax
  mov [edi], eax           ; 再写到内存
  ```

---

### 2️⃣ 算术运算

#### 加法：add
```asm
add eax, 5               ; eax = eax + 5
add eax, ebx             ; eax = eax + ebx
```

#### 减法：sub
```asm
sub eax, 5               ; eax = eax - 5
sub ecx, ebx             ; ecx = ecx - ebx
```

#### 乘法：imul（有符号） / mul（无符号）
```asm
mov eax, 3
imul eax, 4              ; eax = 3 * 4 = 12
```

#### 除法：idiv（有符号） / div（无符号）
```asm
mov eax, 20              ; 被除数放 eax
mov ecx, 3               ; 除数放寄存器
cdq                      ; 符号扩展（必须做！）
idiv ecx                 ; eax = 20 / 3 = 6，edx = 余数
```

---

### 3️⃣ 逻辑运算

```asm
and eax, 0xFF            ; eax = eax & 0xFF（按位与）
or eax, 0x01             ; eax = eax | 0x01（按位或）
xor eax, eax             ; eax = 0（异或，常用来清零）
shl eax, 4               ; eax = eax << 4（左移）
shr eax, 2               ; eax = eax >> 2（右移）
```

---

### 4️⃣ 比较：cmp

**作用**：比较两个值，根据结果设置**标志位**

```asm
cmp eax, 10              ; 比较 eax 是否等于 10
je label                 ; 如果相等，跳转到 label
jne label                ; 如果不相等，跳转到 label
jl label                 ; 如果小于，跳转
jg label                 ; 如果大于，跳转
```

**常用跳转指令**：
- `je`  - 等于 (equal)
- `jne` - 不等于 (not equal)
- `jl`  - 小于 (less)
- `jle` - 小于等于
- `jg`  - 大于 (greater)
- `jge` - 大于等于
- `jmp` - 无条件跳转

---

### 5️⃣ 栈操作：push / pop

**栈的概念**：后进先出 (LIFO)

```
栈顶 → [ ]     ← push 放在这里
       [ ]
       [ ]
       [数据1]
```

```asm
push eax                 ; 把 eax 压入栈
push ebx                 ; 把 ebx 压入栈
pop ecx                  ; 从栈弹出到 ecx
pop edx                  ; 从栈弹出到 edx
```

**重要**：push/pop 必须**配对**！每个 push 必须对应一个 pop。

#### pushad / popad

一次性压入/弹出所有 8 个通用寄存器：

```asm
pushad                   ; 相当于 push eax; push ebx; ... push edi
                         ; 栈上现在有 32 字节（8 个寄存器 × 4 字节）

; 这里可以修改 eax, ebx 等
mov eax, 123
call some_function

popad                    ; 恢复所有寄存器到之前的值
```

---

### 6️⃣ 函数调用

#### call 和 ret

```asm
call function_name       ; 调用函数（自动 push 返回地址到栈）
ret                      ; 返回（自动 pop 返回地址并跳转）
```

**发生的事**：
```asm
call my_func    →  push 下一条指令地址到栈
                    跳转到 my_func

ret             →  pop 栈顶地址（之前 push 的返回地址）
                    跳转回去继续执行
```

#### 函数的标准框架

```asm
my_function:
    push ebp             ; 保存旧帧指针
    mov ebp, esp         ; 建立新帧
    
    ; 函数体...
    mov eax, 42          ; 返回值放 eax
    
    pop ebp              ; 恢复帧指针
    ret                  ; 返回
```

## 第 4 部分：从 C++ 代码看汇编

### C++ 中的 if/else

```cpp
if (a1 < 12) {
    if_flag = a1 / 2 + 1;
} else if (a1 < 24) {
    if_flag = (24 - a1) * a1;
} else {
    if_flag = a1 << 4;
}
```

### 转成汇编：第一步 - 理解逻辑

```
比较 a1 < 12 ?
  YES → if_flag = a1/2 + 1 → 跳到 end
  NO  → 再比较 a1 < 24 ?
        YES → if_flag = (24-a1)*a1 → 跳到 end
        NO  → if_flag = a1 << 4 → 跳到 end
end:
```

### 转成汇编：第二步 - 用标号和条件跳转

```asm
your_if:
    ; 第一个条件：a1 < 12
    mov eax, [a1]        ; eax = a1
    cmp eax, 12          ; 比较 a1 和 12
    jge else_if_2        ; 如果 >= 12，跳到第二个条件
    
    ; 执行：if_flag = a1/2 + 1
    mov eax, [a1]
    cdq
    mov ecx, 2
    idiv ecx             ; eax = a1 / 2
    add eax, 1
    mov [if_flag], eax
    jmp if_end
    
else_if_2:
    ; 第二个条件：a1 < 24
    mov eax, [a1]
    cmp eax, 24
    jge else_3
    
    ; 执行：if_flag = (24 - a1) * a1
    mov eax, 24
    sub eax, [a1]        ; eax = 24 - a1
    imul eax, [a1]       ; eax = (24 - a1) * a1
    mov [if_flag], eax
    jmp if_end
    
else_3:
    ; 执行：if_flag = a1 << 4
    mov eax, [a1]
    shl eax, 4           ; 左移 4 位 = 乘以 16
    mov [if_flag], eax
    
if_end:
    ; 分支结束
```

### C++ 中的 while 循环

```cpp
while (a2 >= 12) {
    int random_num = my_random();
    while_flag[a2 - 12] = random_num;
    --a2;
}
```

### 转成汇编

```asm
your_while:
    mov ecx, [a2]        ; ecx = a2（循环计数器）
    
while_loop:
    cmp ecx, 12          ; a2 >= 12 ?
    jl while_end         ; 如果 < 12，退出循环
    
    ; 调用 my_random()，返回值在 eax
    call my_random
    
    ; while_flag[a2 - 12] = eax
    mov ebx, ecx         ; ebx = a2
    sub ebx, 12          ; ebx = a2 - 12（数组下标）
    mov [while_flag + ebx], eax  ; 写入数组
    
    ; --a2
    dec ecx               ; ecx--
    jmp while_loop        ; 回到循环顶部
    
while_end:
    mov [a2], ecx        ; 更新 a2 的值
```

### C++ 中的函数调用

```cpp
void your_function() {
    for (int i = 0; string[i] != '\0'; ++i) {
        print_a_char(string[i]);
    }
}
```

### 转成汇编

```asm
your_function:
    push ebp
    mov ebp, esp
    
    xor ecx, ecx         ; ecx = 0（计数器 i）
    
loop_start:
    ; 取字符 string[i]
    mov eax, [string]
    mov al, byte [eax + ecx]  ; al = string[i]
    
    cmp al, 0            ; 检查是否为 '\0'
    je loop_end
    
    ; 保存所有寄存器
    pushad
    
    ; 把字符放到栈上作为参数
    push eax             ; 参数：string[i]
    
    ; 调用 print_a_char(string[i])
    call print_a_char
    
    ; 清理栈（恢复 esp）
    add esp, 4           ; 清除 4 字节的参数
    
    ; 恢复寄存器
    popad
    
    ; i++
    inc ecx
    jmp loop_start
    
loop_end:
    pop ebp
    ret
```

## 第 5 部分：完整示例 - 实现分支逻辑

现在让我们完整地写一个分支逻辑的汇编代码。

### 需求

```cpp
if (a1 < 12) {
    if_flag = a1 / 2 + 1;
} else if (a1 < 24) {
    if_flag = (24 - a1) * a1;
} else {
    if_flag = a1 << 4;
}
```

### 完整汇编代码

```asm
your_if:
    ; === 第一个条件检查 ===
    mov eax, dword [a1]   ; eax = a1
    cmp eax, 12           ; 比较 eax 和 12
    jge condition_2        ; >= 12 则跳到条件2
    
    ; === 条件1为真：if_flag = a1/2 + 1 ===
    mov eax, dword [a1]
    mov ecx, 2
    cdq                    ; 符号扩展（除法前必做）
    idiv ecx               ; eax = eax / 2
    add eax, 1             ; eax = eax + 1
    mov dword [if_flag], eax
    jmp if_done            ; 跳到结束
    
condition_2:
    ; === 检查第二个条件 ===
    mov eax, dword [a1]
    cmp eax, 24            ; 比较 eax 和 24
    jge condition_3        ; >= 24 则跳到条件3
    
    ; === 条件2为真：if_flag = (24 - a1) * a1 ===
    mov eax, 24
    mov ecx, dword [a1]
    sub eax, ecx           ; eax = 24 - a1
    imul eax, ecx          ; eax = eax * a1
    mov dword [if_flag], eax
    jmp if_done            ; 跳到结束
    
condition_3:
    ; === 条件3：else分支 ===
    ; if_flag = a1 << 4
    mov eax, dword [a1]
    shl eax, 4             ; eax = eax << 4
    mov dword [if_flag], eax
    
if_done:
    ; 分支执行完毕
```

## 第 6 部分：调试技巧

### 常见错误

| 错误 | 原因 | 解决 |
|------|------|------|
| Segmentation fault | 非法内存访问 | 检查地址计算，确保在有效范围 |
| Undefined symbol | 使用了没定义的变量 | 检查 `extern` 声明 |
| Wrong result | 逻辑错误 | 手动模拟代码执行，追踪寄存器值 |
| Stack overflow | push/pop 不配对 | 数 pushad 和 popad 是否相等 |

### 调试方法

**1. 在关键位置打印数值**

用 `print_a_char` 打印结果：
```asm
mov al, byte [if_flag]   ; 取值到 al
push al
call print_a_char        ; 打印
pop eax
```

**2. 追踪寄存器**

在纸上模拟执行：
```
初始状态：a1 = 15

mov eax, [a1]      → eax = 15
cmp eax, 12        → 15 >= 12，设置标志位
jge condition_2    → 跳转发生（因为 jge 条件满足）
```

**3. 逐行检查**

确认每一步：
- ✅ 数据是否正确读入？
- ✅ 运算是否正确？
- ✅ 内存是否正确写出？
- ✅ 控制流是否正确？

## 第 7 部分：快速参考表

### 常用指令速查

```asm
; 数据移动
mov   dest, src         ; dest = src
lea   dest, [src]       ; dest = src 的地址（不是值）

; 算术
add   dest, src         ; dest += src
sub   dest, src         ; dest -= src
imul  dest, src         ; dest *= src
idiv  src               ; eax = eax / src，edx = 余数
inc   reg               ; reg++
dec   reg               ; reg--

; 逻辑
and   dest, src         ; dest &= src
or    dest, src         ; dest |= src
xor   dest, src         ; dest ^= src
not   reg               ; 按位取反
shl   reg, count        ; reg <<= count
shr   reg, count        ; reg >>= count

; 比较和跳转
cmp   a, b              ; 比较 a 和 b，设置标志位
je    label             ; 相等则跳转
jne   label             ; 不相等则跳转
jl    label             ; 小于则跳转
jle   label             ; 小于等于则跳转
jg    label             ; 大于则跳转
jge   label             ; 大于等于则跳转
jmp   label             ; 无条件跳转

; 栈操作
push  reg               ; 栈上压入
pop   reg               ; 栈上弹出
pushad                  ; 压入所有寄存器
popad                   ; 弹出所有寄存器

; 函数
call  func              ; 调用函数
ret                     ; 返回函数

; 其他
nop                     ; 空指令（什么都不做）
cdq                     ; 符号扩展（除法前）
```
