## 💡 使用建议

### 如何高效学习

1. **不要一次读完**
   - 阅读 → 编码 → 测试 → 再阅读相关章节
   - 这是最好的学习方式

2. **利用快速查找**
   - 遇到特定问题时查看"常见汇编模式.md"或"64位速记卡.txt"
   - 不需要重新读长文档

3. **代码优先**
   - 先复制示例代码运行
   - 再修改代码理解原理
   - 最后从零开始写

4. **调试很重要**
   - 代码出错是正常的
   - 用 QEMU + GDB 调试
   - 看"Assignment 指南"中的调试部分

5. **保存你的代码**
   - 每个 Assignment 的正确版本备份
   - 便于对比和复习

### 命令速查
```bash
# 编译 16位汇编
nasm -f bin code.asm -o code.bin

# 编译 32位汇编（链接 C++）
nasm -f elf32 code.asm -o code.o
gcc -m32 -c main.cpp -o main.o
gcc -m32 code.o main.o -o program

# 编译 64位汇编（链接 C++）
nasm -f elf64 code.asm -o code.o
gcc -m64 -c main.cpp -o main.o
gcc -m64 code.o main.o -o program

# 创建虚拟磁盘
qemu-img create hd.img 10m

# 写入 MBR
dd if=mbr.bin of=hd.img bs=512 count=1 seek=0 conv=notrunc

# 运行 QEMU（等待 GDB）
qemu-system-i386 -hda hd.img -s -S -serial null -parallel stdio

# GDB 调试
gdb
target remote:1234
break *0x7c00
continue
```
