IMPLEMENTATION DETAILS
================================================================================

Thread Structure (thread_t):
  - Size: Fits in PCB_SIZE (4096 bytes)
  - Stack pointer: uint32 *stack
  - Thread ID: uint32 tid (auto-incrementing, starting at 1)
  - Thread name: char name[MAX_PROGRAM_NAME + 1] (max 16 chars)
  - Thread state: ThreadState (enum with 4 states)
  - Priority: uint32 priority (user-defined)
  - Time quantum: uint32 ticks (priority × 10)
  - Execution tracking: uint32 ticks_passed
  - Registers: All 9 x86 general-purpose registers

Stack Frame Layout (x86):
  High Address
    [argument to entry]       (on stack at stack_top - 0)
    [thread_exit return]      (on stack at stack_top - 1)
    [entry function addr]     (on stack at stack_top - 2)
    [ebx/ecx/edx/esi/edi]     (cleared to 0)
    [ebp]                     (cleared to 0)
    [eax]                     (cleared to 0) ← esp points here
  Low Address

Thread Creation Flow:
  1. Allocate thread structure from pool
  2. Allocate stack from pool
  3. Initialize thread metadata
  4. Set up stack frame
  5. Initialize all registers
  6. Return thread_t pointer

Error Handling:
  - Null return on allocation failure
  - Graceful degradation
  - No assertions or crashes
